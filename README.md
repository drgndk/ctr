<h2 id="building">🔨 Installing</h2>

  ```
  git clone https://github.com/drgndk/ctr.git
  cd ctr

  # requires fish shell.
  # Builds the binary and creates a symlink to /usr/bin/ctr
  ./scripts/build.fish

  # or
  cargo build --release
  ```
